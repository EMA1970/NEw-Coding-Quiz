## Description 
This quiz application is built with HTML, CSS, and Javascript. It is designed to test coders knowledge in preparation for interviews or just for fun. The questions topics are on web-development and other full-stack development termnologies. The application uses a browser's local storage to store a user's score from test. 

## Usage 
his application is timed (80secs) and when the timer ends the application will auto close if the test is not finished and the points will not be logged; therefore, the taker must try their best to finish all questions (22 total) before timer ends. All questions are in a multiple choice format, some questions are loosely designed with more than one correct answer. 

## Development 
This application was developed in visual studio.
